Carticon integration. Add a shopping cart to your Wordpress website in minutes.

== Description ==
Carticon integration, easy plug and play system to start selling online.

Add a shopping cart to your Wordpress website in minutes. Connect your Wordpress website to a Carticon account and start selling online with a simple popup window easy to configure and that doesn't change anything in your current website.

**Important**: This plugin needs an account with [Carticon](https://carticon.com) and allows you to easily include that Carticon popup shopping cart in your store.

=Installation=

1. Sign up for a free [Carticon account](https://carticon.com)
2. Copy the embeddable code from Carticon Admin > Settings > Installation
3. Go to Wordpress Admin > Settings > Carticon
4. Paste the embeddable code

=Display the shooping cart=

1. Globally: Check Wordpress Admin > Settings > Carticon > Activate shop
2. Specific page: Use the shortcode `[carticon_shop]` on any page
3. Specific page: Check *Make this page a shop front* on any page (see *Page attributes* section)
